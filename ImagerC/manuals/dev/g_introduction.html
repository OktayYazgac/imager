Introduction to FIPIS
Back to index
 

FIPIS is set of rules and functions definitions that allows developers to write extensions for Futuris Imager. See "General FAQ" for some terms used in this text.

Goal of FIPIS is to extend Futuris Imager various ways (even unknown to program's author). FIPIS is open to any developer willing to write Imager plug-ins. See "Creating Plug-ins".

FIPIS is true plug-in system - main application (imager.exe) do not know about plug-ins until they are loaded. This differs Imager from some other applications - for example IrfanView's plug-in system is fake (main executable needs to be recompiled each time a new plug-in added). Can we call that plug-in system at all?

Futuris Imager is first image viewer with open, true plug-in system.
