Application links are external applications, that launched from Futuris Imager "Tools" menu.

Helpers registered in Windows Registry key "HKEY_CURRENT_USER\Software\Futuris\FIPIS\Apps":

Text for menu=C:\Full\Path\to\GFXviewer.exe

Notes:

  • Add "%1" to the end of path, to get current file path (if file is open) as first command file parameter.
  • Add "%2" to get current file's folder.

Modify this section using installers. It also can be modified by user ("Settings" -> "External Apps" -> "Tool Links" menu).
