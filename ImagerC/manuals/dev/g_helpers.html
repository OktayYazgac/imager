Helpers
Back to index
 

Helpers are external applications, that launched if Imager does not support specific file.

Helpers registered by extension in Windows Registry key "HKEY_CURRENT_USER\Software\Futuris\FIPIS\External":

gfx=C:\Full\Path\to\GFXviewer.exe

Modify this section using installers. It also can be modified by user ("Settings" -> "External Apps" -> "Extension Handlers" menu).
