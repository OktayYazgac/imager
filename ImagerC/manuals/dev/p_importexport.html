Import/Export Plug-ins
Back to index
 

Import and export plug-ins are some sort of extended open/save plug-ins. They are not limited to read bitmaps from files, bitmap source can be virtually anywhere (where you can program it). Import/export plug-ins appear in "Import" and "Export" menus.

To create import plug-in export FIPISimport function. For export plug-ins export FIPISexport function.
