What's New in Futuris Imager 2.6 ?
Back to index 
 
  • fixed broken long file names support (in previous versions "Back" and "Forward" buttons could act incorrectly if current image has file name larger then 8 symbols)
  • file size has much nicer look now
  • changed Windows Explorer document description to "Futuris Imager Document"
  • support for WindowsXP themes is optional now
  • new rewritten File Browser with preview
  • "Create HTML" plug-in now adds DOCTYPE and content encoding to output HTML file
  • "Web View" plug-in resolves some of HTTP errors now
  • large GUI changes and improvements
  • removed Futuris Bitmap plug-in, "Set Wallpaper" plug-in and capture tool - too buggy and useless
