About Futuris Imager
Back to index 
 
Futuris Imager is fast and free graphics viewer for Microsoft Windows (95 / 98 / Me / NT4 / 2000 / XP / 2003).

Supported graphical formats:

  • GIF (CompuServe GIF)
  • JPEG (including progressive)
  • PNG (Portable Network Graphics)
  • MNG (Multiple Network Graphics), JNG (JPEG Network Graphics)
  • BMP (Windows Bitmaps)
  • WBMP (WAP Bitmaps)
  • TIFF
  • Targa
  • PhotoCD
  • Photoshop/PhotoDeluxe
  • Paint Shop Pro
  • and much more...

Major features:

  • Print with preview
  • Advanced printing
  • TWAIN scanners support
  • Built-in file browser
  • Web cams viewing
  • Integration with Filters Unlimited

Futuris Imager features powerful plug-in system, which allows third-party developers to extend Futuris Imager with different types of plug-ins.

Futuris Imager also has some interesting achievements:

  • first image viewer for Windows with open plug-in system
  • first image viewer with WAP bitmap support
