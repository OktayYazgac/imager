About Masks
Back to index 
 

Masks allow you to control what files appear in Imager's open dialog and what files browsed using "<-" and "->" buttons. Type several extensions like this: "*.jpg; *.gif; *.bmp" or type "*.*" to show all files

(but DO NOT type "*.*" into quick browsing mask !!!).
