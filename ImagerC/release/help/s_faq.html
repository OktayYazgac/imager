Q: I'm getting errors while trying to use "File Formats" dialog.

A: "File Formats" dialog may not work properly on Windows NT/2000/XP/2003 systems if you have no administrator or power user privileges.

Q: Can I change settings for JPEG saving via "Save as" dialog ?

A: Sure. Use "Settings" -> "Plug-ins" -> "Configure plug-ins" and select "f_jpeg.dll".

Q: Toolbar buttons in Futuris Imager are strangely small or toolbar buttons are not flat.

A: This is a bug of some versions of Windows Comctl32.dll file (shipped with early releases of Windows95 and also with Internet Explorer 4.0). Download and install latest version from Microsoft's site.

Q: After installing Futuris Imager 2.6 I cannot open any image at all!

A: Rare bug. Solution: in Futuris Imager menu run "Settings" -> "Plug-ins" -> "Scan for Plug-ins". After Imager's restart everything must be fine.

Q: After using filter(s) in Futuris Imager 2.6 transformed image cannot be printed/exported/zoomed. Original image always appears instead of transformed one.

A: After using filters you must "fix" changes to image using "Apply" button in "Filters" panel or "Apply Changes" item in "Edit" menu. You can also use [Ctrl] + [A] keyboard shortcut.

Q: Can I use Futuris Imager in commercial environments ?

A: Yes.

Q: Futuris Imager shows strange colors when I try to view my photos ...

A: On some video cards Imager cannot correctly display HiColor images in 256-color mode. Solution: switch to HiColor or TrueColor modes.

Q: How can I associate Imager with my images ?

A: Go to "Settings" menu and select "File Formats".
