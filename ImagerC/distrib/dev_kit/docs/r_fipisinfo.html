FIPISinfo
Back to index
 

Description:
function that shows information about file.

Definition:
function FIPISinfo(display, path, ext: PChar; app: THandle):integer; stdcall;

Variables:
display - file name/url that should be displayed (if you want to display file path)
path - real file path
ext - extension current plug-in registered for
app - calling process handle

Should return:
1 - if success
0 - if failed

Notes:
n/a
