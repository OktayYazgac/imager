What's new in version 5.2 ?
back to index
 
  • support for various digital camera RAW photo formats
  • support for 48-bit TIFF and PPM images (48 bits per pixel /16 bits per channel)
  • support for embedded colour profiles in TIFF and JPEG images
  • support for HDR (High Dynamic Range) images
  • improved support of 256 colour TGA (Targa) images and 32-bit BMP images with alpha channel
  • multiple file format support fixes
  • speed optimizations in plug-in system
  • updated FreeImage library to version 3.70

released on 25 June, 2005
