Futuris Imager Short Introduction
back to index
 

Futuris Imager is fast and free graphics viewer/manager for Windows with focus on simplicity and unobtrusiveness.

It supports more then 30 graphical formats:

  • GIF, JPEG, JPEG2000
  • PNG, MNG, JNG
  • BMP, WBMP (Wireless Bitmaps)
  • TIFF, Targa, PhotoCD
  • Photoshop/PhotoDeluxe, Paint Shop Pro
  • and many more...

Some other features are:

  • Print with powerful preview
  • TWAIN scanners and cameras support
  • Screen capture
  • HEX viewer
  • Set of filters

Futuris Imager features plug-in system, which allows third-party developers to extend Futuris Imager with different types of plug-ins. For information on plug-in development visit Futuris Imager website.
