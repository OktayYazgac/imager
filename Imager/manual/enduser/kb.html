Keyboard Shortcuts
back to index
 
Shortcut Action
Ctrl + O Open image
Ctrl + S Save image as...
Ctrl + X Close image
Ctrl + P Print image with preview
Ctrl + Z Undo image edits
Ctrl + N Create new image
Ctrl + V Paste image from clipboard
Ctrl + C Copy image to clipboard
Ctrl + F Opens filters manager
Ctrl + I Opens file information
Ctrl + F1 Opens "About" dialog
F1 Opens help
F3 Toggle "Slide Show" toolbar during slide show
F4 Go to random image in folder
F5 Quickly switch between two last opened images
F6 Open Slide Show panel
F7 Open Slide Show panel and start slide show
F11 Switch to/back from Full Screen mode
Esc Exit Imager
Enter Switch between "Fit" and "Normal" displays
Page Up Go to previous image in current folder
Page Down Go to next image in current folder
Shift + Page Up Go to first image in current folder
Shift + Page Down Go to last image in current folder
Space Go to next image in folder
Backspace Go to previous image in folder
Shift + Space Go to last image in folder
Shift + Backspace Go to first image in folder
Insert Go to previous image in current folder
Home Go to next image in current folder
Shift + Insert Go to first image in current folder
Shift + Home Go to last image in current folder
Ctrl + 1 Zoom to 100%
Ctrl + 2 Zoom to 200%
Ctrl + 5 Zoom to 50%
Ctrl + ` (button left to "1") Custom zoom
+ Zoom in
- Zoom out
Shift + Up/Down/Left/ Right Precise scrolling by pages
Up/Down/Left/Right Scrolling
Ctrl + R Rotate image 90° CW
Ctrl + Shift + R Rotate image 90° CCW
   
Mouse wheel Scrolling
Mouse side buttons Go to next/previous image in current folder
