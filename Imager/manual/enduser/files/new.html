What's new in version 5.0 ?
back to index
 
  • new display engine with very fast and memory efficient zooms
  • added options to automatically fit large image into window and to center image in window
  • now it is possible to turn off the scrollbars
  • added multi-level undo for filters
  • added new "Filter" dialog with customisable previews
  • "File Information" dialog now shows EXIF data (including thumbnail), histogram and estimated compression level (for JPEGs)
  • miscellaneous improvements, optimizations and bug fixes
