Back to index
 

Description:
function that saves bitmap from Imager to any imaginable source.

Definition:
function FIPISexport(info: PChar; app: THandle; img: hBitmap):integer; stdcall;

Variables:
info - registration string
app - calling process handle
img - handle of bitmap to export

Should return:
1 - if success
0 - if failed
