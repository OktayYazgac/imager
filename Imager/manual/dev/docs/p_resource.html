FIPIS Plug-ins Resource
Back to index
 

FIPIS plug-ins resource is special type of Windows resource embedded into plug-in DLL. By scanning plug-in dlls for such resources Futuris Imager can understand what dll is plug-in and can be used specific ways.

FIPIS resource is simple INI file contained inside dll. It should be saved in "FIPIS" custom resource type with the name "SPECS" (to create resource in Visual C++ use built-in resource editor, in Delphi or C++ Builder you can use Resource Workshop (from your product CD) or install RXLib 2.75 (or higher) for Project Resource add-on).

For self-explaining INI files see Samples.
