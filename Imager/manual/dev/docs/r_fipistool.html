FIPIStool
Back to index
 

Description:
generic function that can perform any action you want.

Definition:
function FIPIStool(info,path: PChar; app: THandle; img: hBitmap):hBitmap; stdcall;

Variables:
info - registration string
path - current file path
app - calling process handle
img - handle of bitmap

Should return:
bitmap handle - if success
0 - if failed or bitmap transformation not needed

Notes:

  1. Path passed to function is absolute ( "X:\Full\path\image.ext" without quotes).
  2. Returned hBitmap should be released (in Delphi use ReleaseBitmap() method of TBitmap object).
