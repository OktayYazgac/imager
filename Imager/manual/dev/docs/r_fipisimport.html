FIPISimport
Back to index
 

Description:
function that gets bitmap from any source and passes it to Imager.

Definition:
function FIPISimport(info: PChar; app: THandle):hBitmap; stdcall;

Variables:
info - registration string
app - calling process handle

Should return:
bitmap handle - if success
0 - if failed

Notes:

  1. Returned hBitmap should be released (in Delphi use ReleaseBitmap() method of TBitmap object).
