How FIPIS works ?
Back to index
 

When Futuris Imager starts - it reads contents of "HKEY_CURRENT_USER\Software\Futuris\FIPIS" Windows Registry key. It contains list of all plugins available to Imager.

This file automatically generated on Futuris Imager installation. Later it can be updated by user (if it launch "Settings" -> "Plug-ins" -> "Scan for Plug-ins" menu item or by launching plugscan.exe file in same folder with plugins.ini file. Both ways lead to same process, called "Plug-in Scan".

During "Plug-in Scan" special DLL ( plugscan.dll ) will check all DLLs found in plugins subfolder of Imager's folder for special resource section and extract all plug-in information from this section to "HKEY_CURRENT_USER\Software\Futuris\FIPIS" Windows Registry key.

After Imager loaded data from Registry - can start plug-ins registered there by calling exported functions.
