FIPISmultiInit
Back to index
 

Description:
function that initializes multi-page plug-ins before first page queried from Imager.

Definition:
function FIPISmultiInit(filename, ext: PChar; app: THandle):integer; stdcall;

Variables:
filename - filename to load
ext - registered extension
app - calling process handle

Should return:
1 - if success
0 - if failed

Notes:
n/a
