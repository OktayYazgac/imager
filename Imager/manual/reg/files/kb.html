Shortcut Action
Ctrl + O Open image
Ctrl + S Save image as...
Ctrl + X Close image
Ctrl + P Print image with preview (if installed)
Ctrl + Z Undo image transformations
Ctrl + A Apply image transformations
Ctrl + C Copy image to clipboard
Ctrl + V Paste image from clipboard
Ctrl + F Opens filters manager
Ctrl + I Opens file information
Ctrl + F1 Opens "About" dialog
F1 Opens help
F5 Reload image
F6 Open Slide Show panel
F7 Open Slide Show panel and start slide show
F11 Switch to/back from Full Screen mode
Esc Exit Imager
Enter Auto-fit image to window
Insert Go to previous image in folder
Home Go to next image in folder
Shift + Insert Go to first image in folder
Shift + Home Go to last image in folder
Delete Page left
End Page right
Page Up Page up
Page Down Page down
Space Go to next image in folder
Backspace Go to previous image in folder
Shift + Space Go to last image in folder
Shift + Backspace Go to first image in folder
Ctrl + 1 Zoom to 100%
Ctrl + 2 Zoom to 200%
Ctrl + 5 Zoom to 50%
Ctrl + ` (left to 1) Custom zoom
+ Zoom in
- Zoom out
Shift + Up/Down/Left/ Right Precise scrolling
Up/Down/Left/Right Scrolling
