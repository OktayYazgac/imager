What's new in version 4.0 ?
back to index
 
  • added support for Commodore 64 Koala, Amiga IFF, DirectDraw Surface, Sun Raster and X11 Bitmap formats
  • added support for high-resolution PhotoCD files
  • added support for icons (ICO) with multiple images inside
  • partially corrupted PNG files will open without any problems now
  • 19 new filters, including "Twist", "Fish Eye" and "Mosaic"
  • "Crop" plug-in is back (it was mistakenly omitted in 3.0)
  • updated XP-style GUI in main window, favourites and print preview window
  • "Apply Filter" dialog no longer closes after cancelling some filter
  • other minor bug fixes
