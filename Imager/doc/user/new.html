What's new in FuturixImager 5.7 ?
 
  • WIA (Windows Image Acquisition) support
  • Themes (custom toolbar and menu graphics)
  • New "Options" dialog that integrates most FuturixImager settings in single tabbed window
  • Simplified installation of new localizations or themes
  • File formats association with FuturixImager simplified and rare or incompletely supported formats are now hidden by default (this affects only file formats association dialog)
  • All localizations now are Unicode-based internally
  • New XP style theme available (separate download)
  • Faster previews for raw camera files in the "Open" dialog
  • Optimized PNG loading
  • Added ability to load DIB files
  • Fixed issue with delayed localization on program start-up
  • Gradual zooming: 25% -> 50% -> 75% -> 100% -> 125% -> 150% -> 200% -> 300% -> 400%
  • Added keyboard shortcuts for 25%, 75%, 90%, 125%, 150%, 300%, 400% zooms
  • Zooming is now default function for mouse wheel
  • Mouse wheel should automatically switches FuturixImager to normal zoomable mode (if needed)
  • Now FuturixImager respects Windows settings for deleting files (confirmation, Recycle Bin, etc)
  • FuturixImager now detects deletions of images in current folder
  • Improved animated GIF plug-in (rewritten in C++)
  • Added basic help file (it will be expanded in the future)
  • Plug-in system enhancements: plug-in configuration interface, not recommended extensions interface, C++ compatibility - header and sample plug-in are available in SDK
  • Starting from this release FuturixImager will no longer run on Win9x-based operating systems
 
Change log for previous versions is available at FuturixImager website.
