What's new in FuturixImager 5.7.1 ?
 
  • Added option to change smoothing filter for image display (see "Options" dialog)
  • FuturixImager now works fine in limited user accounts (I've made a lot of tests and fixed all found bugs)
  • Improved multi-threading for several GUI operations
  • FreeImage and e-mail plug-ins rewritten in C++ (sources available in SDK)
  • Reorgranized and improved FuturixImager SDK (now including better documentation and complete source of 12 plug-ins)
 
Change log for previous versions available at FuturixImager website.
