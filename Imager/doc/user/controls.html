Keyboard and mouse
 
Keyboard shortcut Action
Enter Switch between display modes (normal or fit)
  or
  Toggle full screen mode
   
+ Zoom in
- Zoom out
   
Page up Go to the previous document in current folder
Backspace  
Back (MM)  
   
Page down Go to the next document in current folder
Space  
Forward (MM)  
   
Shift + Page up Go to the first document in current folder
Shift + Backspace  
Shift + Back (MM)  
   
Shift + Page down Go to the last document in current folder
Shift + Space  
Shift + Forward (MM)  
   
Refresh (MM) Switch between display modes (normal or fit)
Stop (MM) Toggle full screen mode
   
F1 Open FuturixImager help
Ctrl + F1 Show information about FuturixImager version
F2 Launch built-in editor (if possible)
F3 Toggle slide show toolbar (if it was opened before)
F5 Go to the random document in current folder
F6 Open slide show toolbar
F7 Open previously opened document
F11 Toggle full screen mode
   
Arrow keys Document navigation (previous or next document in current folder)
  or
  Scroll document (hold Shift for faster scrolling)
   
Ctrl + 1 Zoom to 100%
Ctrl + 2 Zoom to 200%
Ctrl + 3 Zoom to 300%
Ctrl + 4 Zoom to 400%
Ctrl + 5 Zoom to 50%
Ctrl + 6 Zoom to 25%
Ctrl + 7 Zoom to 75%
Ctrl + 8 Zoom to 125%
Ctrl + 9 Zoom to 150%
Ctrl + 0 Zoom to fit
Ctrl + ` Custom zoom
   
Ctrl + C Copy document to clipboard (if possible)
Ctrl + V Open document from clipboard (if present)
   
Ctrl + I Display information about document
   
Ctrl + K Go to the previous page in multi-page document
Ctrl + L Go to the next page in multi-page document
   
Ctrl + O Open a new document
Ctrl + S Save document in a new format
Ctrl + X Close document
   
Ctrl + M Move document to a new location
Ctrl + T Copy document to a new location
Ctrl + Y Rename document
Delete Delete document (to Recycle Bin if it is enabled)
   
Ctrl + P Print document (with preview before printing)
   
Ctrl + R Rotate document 90° CW
Shift + Ctrl + R Rotate document 90° CCW
   
Ctrl + Z Undo last editing operation
   
Escape Exit FuturixImager
 
MM - means that the keys is available only on special multimedia keyboards
or - means that function of the key can be customized by you
 
Mouse Action
Wheel Document navigation (previous or next document in current folder)
  or
  Scroll document
  or
  Zoom document
   
Ctrl + Wheel Zoom document
  or
  Document navigation (previous or next document in current folder)
   
Shift + Wheel Vertical scroll of the document
   
Shift + Ctrl + Wheel Horizontal scroll of the document
   
Right click Context menu for the document
 
or - means that function of the key can be customized by you
