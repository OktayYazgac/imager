FuturixImager FAQ
 
Q: Does FuturixImager support animated GIFs?
A: Yes, but you need to download separate plug-in from FuturixImager website and if you are using Windows 2000 you also need to download GDI+ redistributable from Microsoft (and put gdiplus.dll into FuturixImager folder).
GDI+ installation is not needed on Windows XP or newer operating systems.
