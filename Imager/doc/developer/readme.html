FuturixImager SDK   Version 5.7
 
In FuturixImager SDK you will find instructions on how to extend Imager and source code of many sample plug-ins. Also provided are localizable strings for translation and sample toolbar graphics.
 
 
Where to find things:
Folder Contents
\localization\ Localizable strings for FuturixImager.
  Source of sample localization DLL (for Visual C++ 2005).
  Archive of old strings (for comparison with current ones).
   
\plugins\ C++ and Delphi headers for plug-in development.
  Sample plug-ins (binaries for most of them are included in latest version of FuturixImager).
  Delphi third-party components used in plug-ins.
   
\theming\ Sample theme source.
