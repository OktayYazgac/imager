Localizing FuturixImager
 
To translate FuturixImager you will need to translate all strings in language_terms.rc (located in this folder) and compile it using provided Visual C++ project.
 
 

Tips for translation:

  1. Translate carefully - if you are not sure in some term or word, leave it untranslated or ask someone else.
  2. Translate only you know both English and your target language very well - there is nothing more annoying then badly translated application.
  3. Be very careful with language_terms.rc file - do not add any additional text there!
  4. Keep language_terms.rc file in Unicode! This is especially important for non-European languages. To work with Unicode files correctly you will need to use editor that supports Unicode text files (Notepad in Windows XP/Vista works fine, also try UltraEdit or Notepad2).
  5. If you need to insert quote into translated string - insert it as double quote ("") - samples:
       1101 		"Translated string with ""quotes"" inside"
   1102			"""Quotes"" inside string"
  6. All strings should be enclosed in quotes - the following string is wrong!
       1101			"String
  7. When updated version of language_terms.rc file is available, be sure to check all changes between strings using some diff tool - I often correct minor mistakes in strings. Great free diff tool is WinMerge.
  8. After you have translated language_terms.rc file - you need to compile it to DLL file to be used in FuturixImager. Using Visual C++ 2008 (either free Express edition or full editions included as part of Visual Studio 2008) open and build "test\Locale.sln" project. By default resulting file will be named fxlangtest.dll (but you can change that in Visual Studio project properties or just rename it afterwards).
  9. Now you need to host this file somewhere - the choice is up to you. After you have found the place to host your translation - send me the link and I will put it to Futurix website!
    Important! I will not link to suspicious websites or websites with extortionate amounts of advertising.
 
 
Strings for previous versions of FuturixImager are available in the "old" folder - this is useful for comparison between versions.
