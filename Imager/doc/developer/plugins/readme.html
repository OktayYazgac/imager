Developing plug-ins for FuturixImager
 
Back in year 2000 Futuris Imager was the first Windows image viewer with plug-in system open to the public - nowadays FuturixImager plug-in system is vastly expanded and more powerful then ever.
 
 

FuturixImager plug-in is a Windows DLL (currently 32-bit only) which exports one required function (FxImgQuery) and one or more optional functions (FxImgOpen, FxImgSave, etc).

Plug-ins are located in FuturixImager folder and found by user when he/she scans for new plug-ins in "Options" dialog.

Plug-ins can be created in Microsoft Visual C++ or Borland Delphi (or in any other native language that supports DLL exports and callbacks).
