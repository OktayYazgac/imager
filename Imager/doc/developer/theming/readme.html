Theming FuturixImager
 
A theme is a set of graphics for toolbars and context menus of FuturixImager.
 
 

Simple step-by-step instructions to create a new theme for FuturixImager:

  1. Copy sample theme (xp) to a new folder.
  2. Replace provided graphics (active.bmp, disabled.bmp, fixed.bmp) with your own (see requirements in a table below).
  3. Edit definition.rc file with Notepad to provide information about your theme and yourself.
  4. Open Theme.sln project in Microsoft Visual C++ 2005 (either free Express edition or full edition included as part of Visual Studio 2005) and build it.
  5. Copy resulting DLL into FuturixImager folder and test it.
 
 
File Description and requirements
active.bmp Provides graphics for toolbars in active state (when buttons are not disabled).
  The colour of the first pixel (top: 1, left: 1) is used as transparent for the whole bitmap.
  All buttons should be of the same size: typically 16x16, 24x24, 32x32 or 48x48 pixels - but it can be any size between 8 and 128 pixels.
  All buttons should be square - button width should be equal to button height and also equal to entire bitmap height.
  Bit depth: 8-bit or 24-bit only.
   
disabled.bmp Provides graphics for toolbars in inactive state (when buttons are disabled).
  Requirements are almost the same as for active.bmp file, except for the size and bit depth which should be exactly the same as in active.bmp - this is important!
   
fixed.bmp Provides graphics for context menu items and a couple of other controls.
  Requirements are almost the same as for active.bmp file, except that the size is fixed to 16x16 pixels.
  This bitmap is for active state of context menu items, no disabled state bitmap needed.
   
definition.rc Contains text information about theme.

There are two options for distribution of your theme:

  1. Via FuturixImager website:
    Send me active.bmp, disabled.bmp, fixed.bmp, and definition.rc files + information about yourself (at least name, but URL or e-mail for users would be nice too). I will compile DLL myself (this is requirement for distribution of themes on my website) and publish it on FuturixImager download page (with your info included).
  2. Via your own website:
    Setup your website with theme download, and send me URL (to the website, not to the download itself).
 
 
Note: new versions of FuturixImager sometimes add new buttons, so your theme may need update in the future (usually new buttons added every 2-3 months).
