Requisitos de Sistema de FuturixImager
 
Requisitos básicos:
  • Windows XP, Windows 2000, o Windows 2003 Server
    FuturixImager también trabaja en Windows XP x64 (como la aplicación 32-bit)
  • Por lo menos 16mb de RAM libre (35 a 60mb para las imágenes grandes)
  • Por lo menos 6mb de espacio libre en la unidad de disco duro
  • 15-bit modo de visualización o mejor (el modo 32-bit recomendado)
 
Requisitos para componentes específicos:
  • Si usted quiere escanear de Imager, su escáner debe soportar WIA o normas TWAIN (el 99% de todos los escáner lo hacen)
