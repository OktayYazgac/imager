What's new in version 4.6 ?
back to index
 
  • completely rewritten TWAIN support: new plug-in includes image preview, compatible with more scanners/cameras and works better with WIA on WindowsXP
  • fixed loading for PhotoCD images that omit largest resolution (6144 x 4096)
  • minor GUI updates for animated and multi-page images
  • included newer GDI+ DLL because of recent Microsoft GDI+ JPEG security bug
    Note: Futuris Imager never used and will never use GDI+ for JPEG images, so users of old version are safe anyway
  • updated LIBMNG to version 1.08

released on October, 10th - 2004
