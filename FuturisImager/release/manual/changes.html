Version 4.6 (October 10, 2004)

  • completely rewritten TWAIN support: new plug-in includes image preview, compatible with more scanners/cameras and works better with WIA on WindowsXP
  • fixed loading for PhotoCD images that omit largest resolution (6144 x 4096)
  • minor GUI updates for animated and multi-page images
  • included newer GDI+ DLL because of recent Microsoft GDI+ JPEG security bug
    Note: Futuris Imager never used and will never use GDI+ for JPEG images, so users of old versions are safe anyway
  • updated LIBMNG to version 1.08

 

Version 4.5 (August 1, 2004)

  • added screen capture tool (capture types: whole screen, active window, rectangle selection, polygon selection)
  • added "HEX Viewer" tool
  • improved toolbar icons look, added several new buttons ("go random", "custom zoom", etc)
  • new slide-show toolbar with more options (and now it can be easily hidden during slide-show)
  • added several print preview options ("proportional", "shrink only large", etc)
  • added "Go Random" navigation mode (naviagtes between random images in current folder) - also added to new slide show toolbar
  • added selected image thumbnail to save dialog
  • "Filters" dialog now shows entire image at once
  • first release of developer's SDK to create plug-ins for Futuris Imager (3.0 and 4.x) available now

 

Version 4.2 (July 19, 2004)

  • Installer now resolves several possible upgrade issues
  • Complete WindowsXP themes support for toolbars
  • Numerous crash fixes (mostly for Windows9x)
  • Added wide-screen and dual-screen wallpaper sizes to "View-as-wallpaper" plug-in
  • Updated FreeImage library to version 3.4.0 - should fix some crashes
  • Fixed compatibility with GDI+ 1.1 (beta)
  • Updated plug-in system - new SDK for developers will be released soon

Version 2.0 (April 16, 2001)

  • Futuris Imager Plug-In System (FIPIS) - plug-in system, that allows developers to extend Futuris Imager functionality.
  • Modularization - reduces all executables and dll sizes, moving common parts into separate dlls. Saves memory by loading required parts only when they actually needed.
  • Improved "Open" dialog with preview.
  • Command line support for /fs (Full Screen) and /print (Print) switches.
  • Support for MNG, JNG formats
  • Improved PNG support - more than 3 times faster than in 1.x versions
  • Save to GIF and PNG added
  • Advanced exporters for GIFs and JPEGs
  • Support for WBMP (WAP bitmaps) open and save
  • Image zooming (several methods available) and auto-fit zoom (hit [ENTER] to use)
  • Print with preview
  • Improved WebOpen (loading images from HTTP servers) - doesn't require MSIE anymore and image format selection is not required (auto-detection)
  • File browser/manager with image preview
  • Ability to create new (empty) images of custom colors
  • Simple (in folder) slide shows
  • 10 filters (image transformations). More will be available soon.
  • Windows9x logo changer
  • Windows (all versions) wallpaper changer
  • "Wallpaper tester" plug-in (view any image as wallpaper without need to set it as wallpaper in Windows)
  • HEX Viewer
  • Screen capture
  • Custom toolbar skins and background colors
