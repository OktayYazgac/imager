Informational Plug-ins
Back to index
 

Informational plug-ins are add-os for Futuris Imager "File Information" dialog box. They must show dialogs with information about file name given.

Informational plug-ins registered per extension. They launched by "Additional Information" button in "File Information" dialog box.

Any kind of information can be displayed.

To create informational plug-in export FIPISinfo function.
