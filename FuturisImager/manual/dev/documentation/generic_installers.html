When you are using installer to install plug-in for Futuris Imager it is very difficult to find Imager's plug-ins folder. Futuris Imager places following information in Windows Registry (in key "HKEY_CURRENT_USER\Software\Futuris\Imager\Paths"):

"Directory" - path to Imager's folder

"PlugsDir" - path to Imager's plugins folder

"Executable" - path to Imager's executable

Notes:

  1. All directory paths are including "\" characters in the end.
  2. To launch plug-ins scanner (to update Imager's plug-ins information) run file "plugscan.exe" in Imager's folder.
  3. If registry settings absent, then you can instruct user to do one of following things:
    - run file "imager.exe" in Imager's folder with "/regdock" param
    - press "Put Imager's INI Path into Registry" button in Imager's "File Formats" dialog box ("Settings" -> "File Formats")
    - note: this information is placed on every Imager start

I recommend using freeware InnoSetup installer. It is very effective.
