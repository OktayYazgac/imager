Toolbar Skins
Back to index
 

Futuris Imager supports "skins" for it's toolbars. "Skins" are 256-color bitmaps, that have ".fsk" extensions and placed in "skins" subfolder of Imager's folder.

Notes:

  1. Top left pixel of image will became transparent as well as all pixels of this color in the image.
  2. All buttons must have dimensions: 16x16 pixels.
  3. User can change skin using Imager's "Select Toolbar Skin" dialog.
  4. Imager's default skin can be found in "Samples".
