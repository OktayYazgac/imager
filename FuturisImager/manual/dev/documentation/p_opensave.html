Open/Save Plug-ins
Back to index
 

Open and save plug-ins are the simplest plug-ins for Futuris Imager. They do straightforward operations: opening file or saving bitmaps to file.

Open and save plug-ins registered per extension. They appear in "Open" and "Save" dialog boxes. Open plug-ins also used for image previews in "Open" dialog.

These plug-ins should be absolutely silent. No error messages or messageboxes should appear. Save plug-ins should use default settings while saving (or settings stored somewhere), but do not ask for options.

To create open plug-in export FIPISopen function. For save plug-ins export FIPISsave function.
