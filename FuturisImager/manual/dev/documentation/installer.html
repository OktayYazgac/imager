Here is instructions to create plug-in package for Futuris Imager's built-in installer.

  1. Create ZIP file containing all plug-in files. NOTE: structure of this ZIP is identical to Imager's plug-in folder (ZIP root = plugins root, subfolders from ZIP = subfolders in plugins folder).
  2. Place, inside ZIP archive, file readme.rtf (or readme.txt) to display information during installation (it is not required).
  3. Change file extension from .zip to .fzp. That's all!

When user launches this file, all contents of .FZP file will be copied to Futuris Imager plugins folder. Readme will be displayed. After that, program will offer user to launch plug-ins scanner to update plug-ins changes.
