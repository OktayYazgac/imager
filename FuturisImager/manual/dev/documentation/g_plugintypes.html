Plug-in Types
Back to index
 

There are a lot of plug-in types for Futuris Imager. Some are defined strictly, others can be used with various purposes. Take a look at full list:

Open Plug-ins
DLLs that perform silent file formats reading.

Save Plug-ins
DLLs that perform silent bitmap saving into misc. file formats.

Import Plug-ins
DLLs that import bitmaps from various sources.

Export Plug-ins
DLLs that export bitmaps.

Filter Plug-ins
Bitmap transformations.

Tool Plug-ins
Generic plug-ins.

Animation Plug-ins
Animation readers.

Multi-page Plug-ins
DLLs that handle multi-page documents.

Helpers
Links to external applications, by extension.

Application links
Links to external applications.
