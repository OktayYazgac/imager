FIPISsave
Back to index
 

Description:
function that save image from hBitmap to local file.

Definition:
function FIPISsave(path, ext: PChar; app: THandle; img: hBitmap):integer; stdcall;

Variables:
path - path to save to
ext - registered extension
app - calling process handle
img - hBitmap to save to file

Should return:
1 - if success
0 - if failed

Notes:

  1. Function shouldn't display any dialogs, dialog boxes or error messages. Completely silent.
  2. Path passed to function is absolute ( "X:\Full\path\image.ext" without quotes).
