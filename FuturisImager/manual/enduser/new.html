What's new in version 4.5 ?

  • added screen capture tool (capture types: whole screen, active window, rectangle selection, polygon selection)
  • added "HEX Viewer" tool
  • improved toolbar icons look, added several new buttons ("go random", "custom zoom", etc)
  • new slide-show toolbar with more options (and now it can be easily hidden during slide-show)
  • added several print preview options ("proportional", "shrink only large", etc)
  • added "Go Random" navigation mode (naviagtes between random images in current folder) - also added to new slide show toolbar
  • added selected image thumbnail to save dialog
  • "Filters" dialog now shows entire image at once
  • first release of developer's SDK to create plug-ins for Futuris Imager (3.0 and 4.x) available now
