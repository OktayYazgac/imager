Q: I'm getting errors while trying to use "File Formats" dialog.

A: "File Formats" dialog will not work on Windows NT4/2000/XP/2003 systems if you have no administrator or power user rights.

Q: Toolbar buttons in Futuris Imager are strangely small or toolbar buttons are not flat.

A: This is a bug of some versions of Windows Comctl32.dll file (shipped with early releases of Windows95 and also with Internet Explorer 4.0). Download and install latest version from Microsoft's site (or update Internet Explorer to 5.01 or higher).

Q: After installing Futuris Imager I cannot open any image at all!

A: Rare bug. Solution: in Futuris Imager menu run "Options" -> "Scan for Plug-ins".

Q: After using filters in Futuris Imager transformed image cannot be printed/exported/zoomed. Original image always appears instead of transformed one.

A: After using filters you must "fix" changes to image using "Apply Changes" item in "Edit" menu. You can also use [Ctrl] + [A] keyboard shortcut.

Q: Can I use Futuris Imager in commercial environments ?

A: Yes.

Q: Futuris Imager shows strange colors when I try to view my photos ...

A: On some video cards Imager cannot correctly display true-color images in 256-color mode. Solution: switch to hi-color or true-color display modes or upgrade display drivers.

Q: How can I associate Imager with my images ?

A: Go to "Settings" menu and select "File Formats".
