Futuris Imager Introduction
back to index
 
Futuris Imager is fast and free graphics viewer/manager for Windows (95 / 98 / 98SE / Me / NT4 / 2000 / XP / 2003).

Supported graphical formats:

  • GIF (including animated ones)
  • JPEG, JPEG2000
  • PNG, MNG, JNG
  • BMP, WBMP (Wireless Bitmaps)
  • TIFF
  • Targa
  • PhotoCD
  • Photoshop/PhotoDeluxe
  • Paint Shop Pro
  • and many more...

Some features:

  • Print with powerful preview
  • TWAIN scanners and cameras support
  • Screen capture
  • HEX viewer

Futuris Imager features plug-in system, which allows third-party developers to extend Futuris Imager with different types of plug-ins.For information on plug-in development visit Futuris Imager website.
