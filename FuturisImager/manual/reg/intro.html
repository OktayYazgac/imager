Futuris Imager Introduction
back to index
 
Futuris Imager is fast and free graphics viewer/manager for Windows (95 / 98 / 98SE / Me / NT4 / 2000 / XP / 2003).

Supported graphical formats:

  • GIF (including animated)
  • JPEG (including progressive)
  • JPEG2000
  • PNG (Portable Network Graphics)
  • MNG (Multiple Network Graphics), JNG (JPEG Network Graphics)
  • BMP
  • WBMP (Wireless Bitmaps)
  • TIFF
  • Targa
  • PhotoCD
  • Photoshop/PhotoDeluxe
  • Paint Shop Pro
  • and many more...

Some features:

  • Print with preview
  • Advanced printing
  • TWAIN scanners support

Futuris Imager features powerful plug-in system, which allows third-party developers to extend Futuris Imager with different types of plug-ins.For information on plug-in development visit Futuris Imager web-site.
